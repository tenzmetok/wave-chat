# wave-chat
WaveChat is a full-stack real-time messaging platform that enables users to communicate instantly through chat rooms, with features such as authentication, online presence tracking, message persistence, and live updates powered by WebSockets.

A full-stack real-time chat platform built with React, Node.js, Socket.io, and PostgreSQL.

## Features

- User Authentication
- Real-time Messaging
- Chat Rooms
- Online Presence
- Typing Indicators
- Message History
- Responsive Design

## Tech Stack

Frontend:
- React
- React Router
- Axios

Backend:
- Node.js
- Express
- Socket.io

Database:
- PostgreSQL

## Architecture

Client → Socket.io → Express → PostgreSQL
